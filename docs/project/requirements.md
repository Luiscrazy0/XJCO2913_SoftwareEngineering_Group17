# 项目需求文档

## 项目概述
电动车租赁管理系统，允许用户发现、预订和租赁电动滑板车，同时提供管理后台进行车辆和订单管理。

## 功能需求

### 用户管理
- [x] 用户注册和登录
- [x] JWT认证和授权
- [x] 用户角色管理（客户、管理员）
- [ ] 用户资料管理
- [ ] 密码重置功能

### 车辆管理
- [x] 车辆列表展示
- [x] 车辆状态管理（可用、不可用、已租）
- [x] 车辆位置信息
- [x] 车辆与站点关联
- [ ] 车辆维护记录
- [ ] 车辆实时位置追踪

### 预订系统
- [x] 在线预订功能
- [x] 多种租赁类型（1小时、4小时、1天、1周）
- [x] 预订状态管理（待支付、已确认、已取消、已完成）
- [x] 续租功能
- [ ] 预订修改功能
- [ ] 预订提醒通知

### 支付系统
- [x] 支付记录管理

### 地图和位置服务

### 管理后台
