# 电动车租赁系统 - 项目文档

## 📋 项目概述
基于NestJS和React的电动车租赁管理系统，支持用户注册、车辆发现、在线预订、支付和管理功能。

## 🛠️ 技术栈
- **前端**: React 18 + TypeScript + Vite + Tailwind CSS
- **后端**: NestJS + TypeScript + Prisma + PostgreSQL
- **认证**: JWT + bcrypt
- **API文档**: Swagger/OpenAPI
- **地图服务**: 高德地图API
- **容器化**: Docker + Docker Compose
